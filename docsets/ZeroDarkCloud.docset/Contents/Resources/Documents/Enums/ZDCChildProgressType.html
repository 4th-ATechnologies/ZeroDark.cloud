ZDCChildProgressType

enum ZDCChildProgressType {}

ZDCProgress is a container for 1 or more children (of type NSProgress). Children can optionally be marked with a special progress type for improved progress reporting.

  • ZDCChildProgressType_MultipartPrep

    When uploading large files, a multipart upload process is used, which splits the large file into several smaller parts, each of which is uploaded separately. This allows the framework to better recover from network interruptions.

    However, this requires a bit of extra prep work (calculating hashes for each chunk). When this occurs, a progress item is added for the ZDCCloudOperation of type ‘MultipartPrep’, which represents the prep work being peformed prior to starting the upload.

    Declaration

    Objective-C

    ZDCChildProgressType_MultipartPrep = 1

    Swift

    case multipartPrep = 1
